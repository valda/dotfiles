Special Keys

RoleKeyDescription
Quit keyC-gキー入力のキャンセル等に用いられます
Help key<f1>任意のキーシーケンスに続けて入力すると、そのキーから始まるキーバインドが一覧表示されます
Escape keyC-qこのキーの後に続けて入力されたキーは Firefox に直接送られます
Start key macro recording<f3>キーマクロの定義を開始します
End key macro recording / Play key macro<f4>キーマクロの定義を終了 ＆ 記録されたキーマクロの再生を行います
Suspension switch key<f2>一時停止状態を切り替えるキーです

Prefix Argument Keys

現在前置引数機能は有効となっています。設定ダイアログからこの機能を無効にすることができます。

KeyDescription
C-u前置引数の入力を開始します。 C-u の後に - を続けると、負の前置引数入力となり、数値が省略された場合は 4 を C-u の数だけ掛け合わせた値となります。
正の前置引数の入力を開始します
C--負の前置引数の入力を開始します
C-M--負の前置引数の入力を開始します
M--負の前置引数の入力を開始します
